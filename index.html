วัดทุก 2 สัปดาห์เพื่อ track ความก้าวหน้า

📊 ตั้งค่า Google Sheets

1
ไปที่ script.google.com → กด New Project
2
ลบโค้ดเดิมทั้งหมด แล้ว วางโค้ดด้านล่างนี้
3
กด Deploy → New deployment → เลือก Web app
4
ตั้ง "Execute as" = Me และ "Who has access" = Anyone → กด Deploy
5
Copy URL ที่ได้ → วางในช่อง "Apps Script URL" ในแอพ → กด บันทึก URL
6
เปิด Google Sheet ที่สร้างขึ้น → Share → "Anyone with the link can view" → Copy ลิงก์ Sheet นั้น
7
วางลิงก์ Sheet ใน แชท Claude แล้วพิมพ์ว่า "ช่วยวิเคราะห์ข้อมูลออกกำลังกายของฉันหน่อย"
✅ ทำครั้งเดียว — หลังจากนี้กด "Sync" แล้ววางลิงก์ในแชทได้เลย ฟรีตลอด
